Load Balancing Algorithms & Protocols

üîÑ Common Load Balancing Algorithms

  • Round Robin: Distribute requests evenly across servers
  • Weighted Round Robin: Assign different weights to servers
  • Least Connections: Route to server with fewest active connections
  • IP Hash: Route based on client IP hash
  • Least Response Time: Route to fastest responding server
  • Health Check Based: Only route to healthy servers

üåê Supported Protocols

  • HTTP/HTTPS: Web traffic (Layer 7)
  • TCP: General TCP connections (Layer 4)
  • UDP: UDP datagrams
  • gRPC: Modern RPC protocol
  • WebSockets: Real-time communication
  • AJP: Apache JServ Protocol (Tomcat specific)
